## Ход выполнения работы

### 1. Request Service

**Bounded Context:** Управление генерацией рецептов

**API:**

| № | Метод | Эндпоинт | Описание |
|---|-------|----------|----------|
| 1 | POST | `/api/requests/generate` | Генерация рецепта из ингредиентов |
| 2 | POST | `/api/requests/{id}/start` | Начало приготовления |
| 3 | POST | `/api/requests/{id}/rate` | Оценка рецепта |
| 4 | GET | `/api/requests/{id}` | Получение рецепта |
| 5 | GET | `/api/requests/history/{user_id}` | История рецептов |

---

### 2. Group Service

**Bounded Context:** Управление пользователями

**API:**

| № | Метод | Эндпоинт | Описание |
|---|-------|----------|----------|
| 1 | POST | `/api/groups/register` | Регистрация пользователя |
| 2 | GET | `/api/groups/{id}` | Получение профиля |
| 3 | PUT | `/api/groups/{id}` | Обновление профиля |
| 4 | GET | `/api/groups/{id}/stats` | Статистика пользователя |

---

### 3. Event Bus (RabbitMQ)

**События:**

| № | Событие | Отправитель | Получатель |
|---|---------|-------------|------------|
| 1 | `RequestCreated` | Request Service | Group Service, Notification Service |
| 2 | `GroupAssigned` | Group Service | Request Service |

**Скриншот RabbitMQ Management:**

```bash
Exchanges:
┌─────────────────────────────────────────────────────────────────┐
│ Name           │ Type    │ Features                             │
├─────────────────────────────────────────────────────────────────┤
│ request.events │ topic   │ Durable                              │
│ group.events   │ topic   │ Durable                              │
└─────────────────────────────────────────────────────────────────┘

Queues:
┌─────────────────────────────────────────────────────────────────┐
│ Name                    │ Messages │ Consumers │ State          │
├─────────────────────────────────────────────────────────────────┤
│ group.queue             │ 0        │ 1         │ Running        │
│ notification.queue      │ 0        │ 1         │ Running        │
└─────────────────────────────────────────────────────────────────┘
```
