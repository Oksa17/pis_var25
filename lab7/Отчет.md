## Ход выполнения работы

### 1. Write Model (Command Model)

**Агрегат:** `Recipe` (рецепт)

**Структура Write Model (нормализованная):**

| Таблица | Поля | Назначение |
|---------|------|------------|
| `recipes` | id, user_id, name, status, source, rating, created_at, updated_at | Основная информация о рецепте |
| `recipe_ingredients` | id, recipe_id, name, expiry_date | Ингредиенты (один ко многим) |
| `recipe_steps` | id, recipe_id, step_order, description, duration_minutes | Шаги приготовления (один ко многим) |
| `users` | id, email, name, role, created_at | Пользователи |

**Схема Write Model:**

```sql
-- Нормализованная схема
CREATE TABLE users (
    id VARCHAR(36) PRIMARY KEY,
    email VARCHAR(255) UNIQUE NOT NULL,
    name VARCHAR(255) NOT NULL,
    role VARCHAR(50) DEFAULT 'user',
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE recipes (
    id VARCHAR(36) PRIMARY KEY,
    user_id VARCHAR(36) NOT NULL REFERENCES users(id),
    name VARCHAR(255) NOT NULL,
    status VARCHAR(50) NOT NULL DEFAULT 'draft',
    source VARCHAR(50) NOT NULL DEFAULT 'ai',
    rating INT CHECK (rating >= 1 AND rating <= 5),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE recipe_ingredients (
    id VARCHAR(36) PRIMARY KEY,
    recipe_id VARCHAR(36) NOT NULL REFERENCES recipes(id) ON DELETE CASCADE,
    name VARCHAR(100) NOT NULL,
    expiry_date DATE
);

CREATE TABLE recipe_steps (
    id VARCHAR(36) PRIMARY KEY,
    recipe_id VARCHAR(36) NOT NULL REFERENCES recipes(id) ON DELETE CASCADE,
    step_order INT NOT NULL,
    description TEXT NOT NULL,
    duration_minutes INT
);
```

### 3. Event-Driven Sync

**События CQRS:**

| № | Событие | Триггер | Действие в Read Model |
|---|---------|---------|----------------------|
| 1 | `RecipeGeneratedEvent` | Генерация рецепта | Создать запись в `recipe_views` |
| 2 | `RecipeStatusChangedEvent` | Изменение статуса | Обновить статус в `recipe_views` |
| 3 | `RecipeRatedEvent` | Оценка рецепта | Обновить рейтинг в `recipe_views` |
| 4 | `RecipeDeletedEvent` | Удаление рецепта | Удалить запись из `recipe_views` |

**Код проекции (Projection Handler):**

```python
# infrastructure/event_handlers/recipe_projection.py
from datetime import datetime
from src.domain.events.recipe_generated_event import RecipeGeneratedEvent
from src.domain.events.recipe_status_changed_event import RecipeStatusChangedEvent
from src.domain.events.recipe_rated_event import RecipeRatedEvent
from src.domain.events.recipe_deleted_event import RecipeDeletedEvent
from src.infrastructure.db.read_models import ReadModelSession
from src.infrastructure.db.models_read import RecipeViewModel


class RecipeProjection:
    """Обработчик событий для обновления Read Model (CQRS)"""
    
    def __init__(self, read_db_session: ReadModelSession):
        self._db = read_db_session
    
    def handle_recipe_generated(self, event: RecipeGeneratedEvent):
        """RecipeGeneratedEvent → создать RecipeView"""
        view = RecipeViewModel(
            id=event.recipe_id,
            user_id=event.user_id,
            user_name=event.user_name,
            recipe_name=event.recipe_name,
            status="generated",
            source=event.source,
            rating=None,
            ingredients_list=", ".join(event.ingredient_names),
            steps_count=event.steps_count,
            total_duration=event.total_duration,
            created_at=datetime.now(),
            updated_at=datetime.now()
        )
        self._db.add(view)
        self._db.commit()
        print(f"✅ Read Model обновлена: создан рецепт {event.recipe_id}")
    
    def handle_recipe_status_changed(self, event: RecipeStatusChangedEvent):
        """RecipeStatusChangedEvent → обновить статус в RecipeView"""
        view = self._db.query(RecipeViewModel).filter_by(id=event.recipe_id).first()
        if view:
            view.status = event.new_status
            view.updated_at = datetime.now()
            self._db.commit()
            print(f"✅ Read Model обновлена: статус {event.recipe_id} → {event.new_status}")
    
    def handle_recipe_rated(self, event: RecipeRatedEvent):
        """RecipeRatedEvent → обновить рейтинг в RecipeView"""
        view = self._db.query(RecipeViewModel).filter_by(id=event.recipe_id).first()
        if view:
            view.rating = event.rating
            view.updated_at = datetime.now()
            self._db.commit()
            print(f"✅ Read Model обновлена: рейтинг {event.recipe_id} → {event.rating}★")
    
    def handle_recipe_deleted(self, event: RecipeDeletedEvent):
        """RecipeDeletedEvent → удалить RecipeView"""
        view = self._db.query(RecipeViewModel).filter_by(id=event.recipe_id).first()
        if view:
            self._db.delete(view)
            self._db.commit()
            print(f"✅ Read Model обновлена: удалён рецепт {event.recipe_id}")
```
